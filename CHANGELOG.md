# Changelog

本文档记录 VidBoost 的版本更新历史。

整理依据：

- 已发布的 GitHub Releases 公告
- 各版本 tag 区间内的真实 commits
- 仓库内历史 release 公告草稿与发布说明文档

## [1.8.1] - 2026-05-29

### Fixed
- 修复部分自定义 SVG 图标因内部 `fill` / `stroke` 固定颜色导致关闭态未完全变灰的问题。

## [1.8.0] - 2026-05-22

### Added
- 新增 YouTube 字幕状态记忆能力，打开其他视频时可恢复 CC 字幕开关状态。
- 新增字幕状态记忆范围设置，支持“全部视频共用”与“按频道分别记住”两种模式。
- 为 YouTube 视频下方进度条粗细设置新增直观预览，方便在 popup 中直接判断显示效果。

### Changed
- 重构 popup 功能选项页的通用设置样式，统一设置面板、字段容器、分段控件、标签列表与状态提示的视觉语言。
- 重排 YouTube 自渲染字幕样式面板，优化字体、颜色、效果和已导入字体管理的空间利用与信息层次。
- 优化 YouTube 会员视频屏蔽面板，将“全部屏蔽 / 仅屏蔽 / 仅允许”改为更清晰的分段选择，并重做频道名单编辑交互。
- 优化 Bilibili 自动打开中文字幕与指定画质相关面板，统一“全部视频 / 指定 UP 主”等范围切换和名单输入体验。
- 调整视频下方进度条粗细设置的表达方式，让数值调节与实际显示效果更容易对应。

### Fixed
- 修复禁用状态下部分 toggle 仍可通过点击或键盘触发的问题。
- 修复折叠面板展开动作与主开关状态切换耦合导致的误操作问题。
- 修复 YouTube 自渲染字幕在选择已导入字体后不会立即应用到当前样式的问题。

## [1.7.0] - 2026-05-15

### Added
- 新增 YouTube 视频下方进度条能力，在播放器底部提供一条精简进度提示，并支持调节进度条粗细。
- 新增 YouTube 底部进度条对应的 popup 配置入口，沿用现有 YouTube 设置区的主开关与展开式交互。
- 新增 YouTube 自渲染字幕与底部进度条相关功能截图，用于 README 和后续发布展示。

### Changed
- 调整 YouTube 底部进度条显示策略，仅在原生控制条自动隐藏后显示，原生控件重新出现时自动隐藏，直播场景下自动关闭该进度条。
- 补充 README 中关于 YouTube 自渲染字幕与视频下方进度条的功能说明，并同步更新展示截图。
- 补充 Firefox feature coverage 与 package 校验规则，确保新的 YouTube 底部进度条模块进入 Firefox 构建产物。
- 优化 release notes 生成逻辑：当 CHANGELOG 中存在当前版本条目时，优先直接使用该条目生成发布说明，并统一将完整更新链接指向当前 release 页面。
- 将版本号统一升级至 `1.7.0`。

## [1.6.3] - 2026-05-14

### Added
- 新增 GitHub Actions 自动发布流程，推送 `v*` tag 后可自动构建并创建 GitHub Release。
- 新增版本一致性校验脚本，保证 `package.json`、`package-lock.json`、`public/manifest.json` 保持同步。
- 新增自动 release notes 生成脚本，可基于 tag 区间整理版本变更。
- 新增 Chrome Release 打包脚本，统一生成 GitHub Release 附件。
- 新增仓库级 contributor guide 与 GitHub Actions 发版说明文档。

### Changed
- 优化 YouTube 自渲染字幕的全屏缩放逻辑，避免全屏时字体线性放大过头。
- 优化 YouTube 字幕拖拽手柄尺寸策略，避免跟随字幕一起过度放大。
- popup 中的版本显示兜底逻辑改为通用默认值，降低版本漂移风险。

## [1.6.2] - 2026-05-07

### Added
- 继续完善 YouTube 自渲染字幕与播放器增强体验，让字幕样式面板、显示效果和相关控件更稳定。

### Fixed
- 修复 `屏蔽会员视频` 失效问题，恢复 YouTube 会员视频过滤能力。

### Changed
- 优化 YouTube 自渲染字幕与 CDN 控件显示体验，让字幕样式与相关控件交互更稳定。

## [1.6.1] - 2026-04-25

### Added
- 新增 YouTube 自渲染字幕能力，由 VidBoost 按当前播放进度自行渲染字幕，重点改善高倍速场景下的字幕同步体验。
- 新增字幕位置拖动能力，支持直接调整字幕显示位置。
- 新增字幕样式自定义能力，支持字体、字号、字重、文字颜色、透明度、背景颜色、背景透明度与背景圆角。
- 新增字符边缘效果，支持描边、投影、浮雕、压印，并支持多效果叠加。
- 新增字幕字体导入能力，支持 `.ttf / .otf / .woff / .woff2` 导入后直接用于字幕渲染。

### Fixed
- 修复字幕拖动不可用的问题，并优化拖动手柄的显示 / 隐藏交互。
- 优化播放器点击命中逻辑，减少 YouTube 场景下 `fast pause` 的误触问题。
- 修复部分暂停逻辑的误触场景。

### Changed
- 增强 YouTube 字幕字体自定义能力。
- 优化字幕边缘样式渲染与样式面板交互。
- 增加自渲染失败时自动回退到 YouTube 原生字幕的稳定性逻辑。
