# Changelog

本文档记录 VidBoost 的版本更新历史。

整理依据：

- 已发布的 GitHub Releases 公告
- 各版本 tag 区间内的真实 commits
- 仓库内历史 release 公告草稿与发布说明文档

## [1.6.3] - 2026-05-14

### Added
- 新增 GitHub Actions 自动发布流程，推送 `v*` tag 后可自动构建并创建 GitHub Release。
- 新增版本一致性校验脚本，保证 `package.json`、`package-lock.json`、`public/manifest.json` 保持同步。
- 新增自动 release notes 生成脚本，可基于 tag 区间整理版本变更。
- 新增 Chrome Release 打包脚本，统一生成 GitHub Release 附件。
- 新增仓库级 contributor guide 与 GitHub Actions 发版说明文档。

### Changed
- 优化 YouTube 自渲染字幕的全屏缩放逻辑，避免全屏时字体线性放大过头。
- 优化 YouTube 字幕拖拽手柄尺寸策略，避免跟随字幕一起过度放大。
- popup 中的版本显示兜底逻辑改为通用默认值，降低版本漂移风险。

## [1.6.2] - 2026-05-07

### Added
- 继续完善 YouTube 自渲染字幕与播放器增强体验，让字幕样式面板、显示效果和相关控件更稳定。

### Fixed
- 修复 `屏蔽会员视频` 失效问题，恢复 YouTube 会员视频过滤能力。

### Changed
- 优化 YouTube 自渲染字幕与 CDN 控件显示体验，让字幕样式与相关控件交互更稳定。

## [1.6.1] - 2026-04-25

### Added
- 新增 YouTube 自渲染字幕能力，由 VidBoost 按当前播放进度自行渲染字幕，重点改善高倍速场景下的字幕同步体验。
- 新增字幕位置拖动能力，支持直接调整字幕显示位置。
- 新增字幕样式自定义能力，支持字体、字号、字重、文字颜色、透明度、背景颜色、背景透明度与背景圆角。
- 新增字符边缘效果，支持描边、投影、浮雕、压印，并支持多效果叠加。
- 新增字幕字体导入能力，支持 `.ttf / .otf / .woff / .woff2` 导入后直接用于字幕渲染。

### Fixed
- 修复字幕拖动不可用的问题，并优化拖动手柄的显示 / 隐藏交互。
- 优化播放器点击命中逻辑，减少 YouTube 场景下 `fast pause` 的误触问题。
- 修复部分暂停逻辑的误触场景。

### Changed
- 增强 YouTube 字幕字体自定义能力。
- 优化字幕边缘样式渲染与样式面板交互。
- 增加自渲染失败时自动回退到 YouTube 原生字幕的稳定性逻辑。

## [1.5.0] - 2026-04-02
